# **1-mini-robocon-machine**
一年生の部内ロボコンで製作した機体についての説明です。

## ルール
フィールド上に散らばったカラーボールを拾い、300mmほど離れた場所に設置された高さ200mmほどのゴミ箱に入れる。
1vs1であり、共通フィールドで競技を行う。
